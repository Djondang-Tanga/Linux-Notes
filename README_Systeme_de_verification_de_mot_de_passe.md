# Systeme_de_verification_de_mot_de_passe.py

Script Python qui vérifie si un mot de passe respecte les critères de longueur.

## Fonctionnement

L'utilisateur entre un mot de passe, le programme vérifie sa longueur et affiche un message.

## Critères

- Moins de 8 caractères → trop court
- Entre 8 et 20 caractères → valide
- Plus de 20 caractères → trop long

## Utilisation

```bash
python3 Systeme_de_verification_de_mot_de_passe.py
```

## Exemple

```
Entrez votre mot de passe : monmotdepasse
Le mot de passe est valide.
```
