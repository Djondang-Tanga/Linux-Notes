# Semaine 1 — Variables

Mon premier script Python. J'ai appris à stocker des informations 
dans des variables et à les afficher dans une phrase.

## Ce que j'ai fait
J'ai créé deux variables : mon prénom et mon âge, puis je les ai 
affichés avec une phrase en utilisant les f-strings.

## Ce que j'ai compris
- Une variable c'est comme une boîte qui garde une information
- `str` pour du texte, `int` pour un nombre entier
- Les f-strings permettent de coller des variables dans une phrase facilement
