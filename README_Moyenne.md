# Moyenne.py

Script Python qui calcule la moyenne de trois notes saisies par l'utilisateur.

## Fonctionnement

L'utilisateur entre trois notes, le programme calcule et affiche leur moyenne.

## Utilisation

```bash
python3 Moyenne.py
```

## Exemple

```
Entrez la première note : 12
Entrez la deuxième note : 15
Entrez la troisième note : 18
La moyenne des trois notes est : 15.0
```
