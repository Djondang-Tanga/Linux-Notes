# Travaille_sur_boucle.py

Script Python qui vérifie si l'achat d'un ordinateur est possible selon le solde du portefeuille.

## Fonctionnement

Le programme compare le prix de l'ordinateur avec le solde disponible et affiche si l'achat est possible ou non.

## Valeurs par défaut

- Portefeuille : 5000 €
- Prix ordinateur : 1200 €

## Utilisation

```bash
python3 Travaille_sur_boucle.py
```

## Exemple

```
L'ordinateur est trop cher.
L'ordinateur est abordable.
```
