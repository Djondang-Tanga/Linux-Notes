# Valeur_portefeuille.py

Script Python qui calcule la nouvelle valeur d'un portefeuille après l'achat d'un téléphone.

## Fonctionnement

L'utilisateur entre la valeur de son portefeuille, le programme déduit le prix du téléphone et affiche le solde restant.

## Valeurs par défaut

- Prix du téléphone : 500 €

## Utilisation

```bash
python3 Valeur_portefeuille.py
```

## Exemple

```
Entrez la valeur de votre portefeuille : 2000
La nouvelle valeur de votre portefeuille après l'achat du téléphone est : 2500.0
```
